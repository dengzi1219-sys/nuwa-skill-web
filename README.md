# nuwa-skill-web
## 🌌 女娲 · 人格蒸馏 v1.4

### ✨ 新增功能
- **人格分析报告**：蒸馏完成后自动生成一份冷静的叙事型人格分析报告（Tab5）
- **事实核查（审核Agent）**：可选开启，独立模型对比语料和蒸馏结果，检测事实不一致（Tab6）
- **对话者标注**：上传多文件时可标注每份文件的对话双方，帮助AI区分不同关系中的表现

### 🔧 改进
- 模型选项精简为 `deepseek-v4-flash` 和 `deepseek-v4-pro`
- 缺口分析模型固定为 `deepseek-v4-flash`（省钱）
- 审核模型固定为 `deepseek-v4-flash`（省钱）
- 完整输出 Tab 现可捕获流式推理过程（thinking 阶段）

### 🛠 修复
- 上传文件 + 粘贴文本共存（不互斥）
- 缺口分析页支持上传文件（不只是贴文本）
- 修复文件名含特殊字符导致保存失败的问题
- 修复部分场景下 Markdown 预览为空的问题

### 📦 文件结构
app.py # 主程序
prompts.py # 提示词模块
checker.py # 审核模块（新增）
reports.py # 报告生成模块（新增）
saves/ # 自动保存目录（不上传）


### 🚀 使用方法
1. 打开网页（Streamlit部署后）
2. 输入人物姓名 + 上传聊天记录（.txt / .md / .json）
3. 可选：标注对话者身份
4. 勾选「同时生成人格分析报告」和「蒸馏后自动核查」（可选）
5. 点击「开始分析」
6. 在结果页查看 JSON、Markdown、人格分析报告、事实核查
